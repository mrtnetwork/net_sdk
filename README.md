## net_sdk

**net_sdk** is a Rust networking bridge focused on providing a unified way to communicate over multiple internet protocols **through Tor or direct connections**.

### Features

- Unified API for multiple transports
- Supported protocols:
  - HTTP
  - TCP
  - TLS
  - gRPC
- Optional Tor routing for all protocols
- Async runtime support
- Designed for embedding in cross-platform SDKs
